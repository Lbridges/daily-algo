Supervised Learning

In regards to machine learning, supervised learning is distinguished by given input and output values. Historic data holds correct answers and the algorithm is designed to find them in new data.

Regression Technique

Regression fits the data

Classification Technique

Classification separates the data. In this form, data is already labeled

Three main supervised learning tasks

  • Binary Classification -two categories of classification
  • Multiclass Classification - more than two categories
  • Regression - predicts continuous values (housing prices are a common example)

This is why we divide supervised learning between classification and regression.

Common Use Cases of Supervised Learning

  • Bioinformatics
  • Database Marketing
  • Handwriting Recognition
  • Pattern Recognition
  • Speech Recognition
  • Information Extraction
